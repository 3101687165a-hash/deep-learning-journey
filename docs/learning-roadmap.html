知识体系

深度学习基础

  • 线性回归 → Logistic 回归 → 前馈网络 (MLP)
  • 反向传播: 链式法则，计算图，自动求导
  • 激活函数: ReLU / GELU / Swish 及其梯度特性
  • 损失函数: MSE / CrossEntropy / NLL

训练技巧 & 正则化

  • Batch Normalization: 内部协变量偏移
  • Dropout: 集成学习的近似
  • L1/L2 正则化: 稀疏性 vs 权重衰减
  • Data Augmentation: 数据增强的正则化效果

卷积神经网络

  • Conv2d: 局部感受野, 参数共享, 平移不变性
  • Pooling: MaxPool / AvgPool / Adaptive Pool
  • 经典架构: VGG → ResNet → EfficientNet
  • 残差连接: 梯度高速公路

循环神经网络

  • RNN: 时序建模, BPTT, 梯度消失
  • LSTM: 门控机制 (遗忘门/输入门/输出门)
  • GRU: 简化的门控, 参数量更少
  • 双向 RNN: 同时利用上下文信息

Transformer 架构

  • Self-Attention: 全局依赖捕捉
  • Multi-Head: 多子空间并行注意力
  • Positional Encoding: sin/cos vs 可学习
  • LayerNorm → 适用于变长序列的归一化
  • 残差连接 + FFN: 深层网络的稳定性

优化与工程

  • SGD / Adam / AdamW: 优化器原理与选择
  • 学习率调度: Step / Cosine / Noam / Warmup
  • 梯度裁剪: 防止梯度爆炸
  • 混合精度训练 (AMP)
  • DDP: 分布式数据并行
